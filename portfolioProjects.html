                  TMDb

For TMDb, I utilized React, MongoDB, Spring Boot, and Bootstrap to create a user-friendly website for movie enthusiasts. I focused on creating an aesthetically pleasing design for the project, some features include user authentication, a search feature using the MovieDB API, and personalized recommendations.

               mealShare

For this project, I utilized design tools like Figma to create a mockup and design the user workflow for an app called mealShare. I then created aesthetically pleasing web pages using HTML and CSS, with a focus on simplifying the user experience. Finally, I designed the database on DBDiagram and set it up on Cloud Firestore.

         TrojanExpeditions

For TrojanExpeditions, I used Vue.js, Spring Boot, and SQL to create a website that helps USC students find the best study spots on campus. The website includes a rating system based on filters like quietness and aesthetics, a student forum, and an interactive map.

                SalStocks

For SalStocks, I utilized Java to parse JSON and CSV files to store stock database and future stock trades, checking for valid input along the way. I then implemented multi-threading to simulate stock trades by stock brokers set by user-inputted files.
